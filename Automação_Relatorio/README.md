# Automação de Relatório Diário com Python

Automação que:

- Abre o Chrome automaticamente
- Baixa relatório de vendas
- Calcula faturamento com Pandas
- Envia email automático via Gmail

## Tecnologias usadas

- Python
- PyAutoGUI
- Pandas
- OpenPyXL
- Pyperclip

## Observações

As coordenadas do PyAutoGUI podem variar conforme resolução da tela.

Projeto criado para estudo de automação de processos.
